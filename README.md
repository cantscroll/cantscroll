# cantscroll

Coding-agent-aware phone blocker for developers.

This repo hosts release artifacts. Source code is at
[github.com/usamaunais/focus-lock](https://github.com/usamaunais/focus-lock) (private).

## Install

```bash
brew tap cantscroll/cantscroll
brew install cantscroll
```
